Accessibility

Accessibility Statement

When designing this site accessibility was taken into account all images have been given an alt tag so that non-text content is easily interpretable via a screen reader. In addition, the HTML used is well formed, which means the user can predictably jump around the page using access keys. Semantic elements have also been used to help screen reader software and those using access keys quickly access parts of the site which are most of relevance to them.
